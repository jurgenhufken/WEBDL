# Screen Tools Firefox Extensie

Een Firefox extensie waarmee je gemakkelijk screenshots kunt maken en je scherm kunt opnemen met audio.

## Functionaliteiten

- Screenshot functie: maak direct een screenshot van het zichtbare tabblad
- Scherm opname functie: neem je scherm op met audio
- Visuele indicator tijdens opname
- Automatisch opslaan naar je downloads map

## Installatie

1. Download de extensie bestanden naar je computer
2. Open Firefox en ga naar `about:debugging#/runtime/this-firefox`
3. Klik op "Tijdelijke Add-on laden"
4. Navigeer naar de gedownloade bestanden en selecteer het `manifest.json` bestand
5. De extensie is nu geïnstalleerd voor deze sessie

Voor permanente installatie:
1. Zip de bestanden (hele map firefox-screen-tools)
2. Hernoem het bestand naar `.xpi` extensie
3. Ga naar `about:addons` in Firefox
4. Klik op het tandwiel icoon en selecteer "Install Add-on From File"
5. Selecteer het .xpi bestand

## Gebruik

1. Klik op het extensie icoon in de Firefox toolbar om het menu te openen
2. Klik op "Screenshot" om direct een screenshot te maken van het huidige tabblad
3. Klik op "Opnemen" om je scherm op te nemen met audio
4. Tijdens opname verschijnt er een rode indicator om aan te geven dat de opname bezig is
5. Klik op "Stoppen" om de opname te beëindigen

Alle bestanden worden automatisch opgeslagen in je standaard downloads map.

## Vereisten

- Firefox 57 of hoger

## Let op

Voor schermopname heb je toestemming nodig van de browser. Firefox zal om deze toestemming vragen wanneer je de opnamefunctie gebruikt.
